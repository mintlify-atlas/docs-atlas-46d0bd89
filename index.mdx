The .NET MAUI Community Toolkit is a collection of common elements for development with .NET MAUI that developers tend to replicate across multiple apps. It simplifies and demonstrates common developer tasks when building apps with .NET MAUI for iOS, Android, macOS, Windows, and Tizen.

## Quick install

<Steps>
  <Step title="Install the NuGet packages">
    Add the packages you need to your .NET MAUI project:

    ```bash
    dotnet add package CommunityToolkit.Maui
    dotnet add package CommunityToolkit.Maui.MediaElement
    dotnet add package CommunityToolkit.Maui.Camera
    dotnet add package CommunityToolkit.Maui.Maps
    ```
  </Step>
  <Step title="Initialize in MauiProgram.cs">
    Call `UseMauiCommunityToolkit()` on your `MauiAppBuilder`:

    ```csharp MauiProgram.cs
    using CommunityToolkit.Maui;

    public static class MauiProgram
    {
        public static MauiApp CreateMauiApp()
        {
            var builder = MauiApp.CreateBuilder();
            builder
                .UseMauiApp<App>()
                .UseMauiCommunityToolkit();

            return builder.Build();
        }
    }
    ```
  </Step>
  <Step title="Add the XAML namespace">
    Reference toolkit controls and behaviors in your XAML pages:

    ```xml
    xmlns:toolkit="http://schemas.microsoft.com/dotnet/2022/maui/toolkit"
    ```
  </Step>
  <Step title="Use the toolkit">
    You're ready to use controls, behaviors, converters, and more across your app. See the [Controls](/controls/popup) section for detailed usage examples.
  </Step>
</Steps>
